### Instagram

#### Content Formats
| Format | Aspect Ratio | Resolution | Duration | File Size |
|--------|-------------|------------|----------|-----------|
| Feed Post (square) | 1:1 | 1080×1080 | — | 30MB photo |
| Feed Post (portrait) | 4:5 | 1080×1350 | — | 30MB photo |
| Feed Post (landscape) | 1.91:1 | 1080×608 | — | 30MB photo |
| Carousel | 1:1 or 4:5 | 1080×1080/1350 | — | 10 slides max |
| Reels | 9:16 | 1080×1920 | 15s-90s (up to 15min) | 4GB video |
| Stories | 9:16 | 1080×1920 | Up to 60s | 4GB video |
| Live | 9:16 | 720p min | Up to 4 hours | — |
| IGTV / Long video | 9:16 or 16:9 | 1080p | Up to 60 min | 3.6GB |

#### Caption Specs
| Element | Limit |
|---------|-------|
| Caption length | 2,200 characters |
| Hashtags | 30 max (recommend 3-5) |
| Mentions | 20 per post |
| Bio link | 1 (or 5 with link feature) |

#### Algorithm Signals (current)
- Reels prioritised in feed and Explore
- Watch time & replays are key ranking signals
- Shares (via DM) heavily weighted
- Saves signal high-value content
- Comments > Likes for engagement quality

### TikTok

#### Content Formats
| Format | Aspect Ratio | Resolution | Duration | File Size |
|--------|-------------|------------|----------|-----------|
| Standard video | 9:16 | 1080×1920 | 15s-10min | 287MB (iOS) / 72MB (Android) |
| Photo mode | 9:16 | 1080×1920 | — | Up to 35 photos |
| LIVE | 9:16 | 720p+ | Min 1000 followers | — |
| Stories | 9:16 | 1080×1920 | Up to 15s | — |

#### Caption Specs
| Element | Limit |
|---------|-------|
| Caption length | 4,000 characters |
| Hashtags | No hard limit (3-5 recommended) |
| Sounds | Must be non-copyrighted for commercial |

#### Algorithm Signals
- Completion rate is #1 signal
- Loop-ability (watching again)
- Shares > Saves > Comments > Likes
- First 1-3 seconds critical (hook)
- Trending sounds boost discovery
- Niche consistency helps long-term

### YouTube

#### Content Formats
| Format | Aspect Ratio | Resolution | Duration | File Size |
|--------|-------------|------------|----------|-----------|
| Standard video | 16:9 | 1920×1080 (min) / 4K | No hard limit | 256GB or 12 hours |
| Shorts | 9:16 | 1080×1920 | Up to 60s | — |
| Live | 16:9 | 1080p+ | No limit | — |
| Premiere | 16:9 | Same as standard | Same | Same |

#### Thumbnail
- 1280×720 (16:9)
- Under 2MB
- JPG, GIF, or PNG
- CTR benchmark: 4-10%

#### Caption / Description
| Element | Limit |
|---------|-------|
| Title | 100 characters (recommend 60) |
| Description | 5,000 characters |
| Tags | 500 characters total |
| Chapters | Timestamps in description |
| Cards | Up to 5 per video |
| End screens | Last 5-20 seconds |

#### Algorithm Signals
- Click-through rate (thumbnail + title)
- Average view duration / retention
- Session time (keeps viewers on YouTube)
- Engagement: likes, comments, shares
- Upload consistency

### Twitter / X

#### Content Formats
| Format | Specs |
|--------|-------|
| Tweet | 280 characters |
| Image | 1200×675 (16:9) or 1080×1080, up to 4 images, 5MB each |
| Video | 1280×720 min, up to 2:20 (or 10 min for some), 512MB max |
| Spaces (audio) | Live audio, unlimited duration |
| Threads | Unlimited tweets |

#### Engagement Tips
- Threads outperform single tweets for reach
- Visual tweets get 150%+ more engagement
- Quote tweets > retweets for algorithm
- First hour engagement is critical

### LinkedIn

#### Content Formats
| Format | Specs |
|--------|-------|
| Text post | 3,000 characters |
| Image | 1200×627 (landscape) or 1080×1080 | 
| Video | 3s-10min, 5GB max, 16:9 or 1:1 or 9:16 |
| Document / carousel | PDF upload, up to 300 pages |
| Newsletter | Long-form, subscribers notified |
| Live | Via third-party streaming tools |
| Article | Unlimited length |

#### Algorithm Signals
- Dwell time (reading/viewing time)
- Comments > Reactions
- First 90 minutes critical
- Native content > external links
- Document posts get high distribution

### Disclosure / Sponsored Content Tools

| Platform | Native tool | How to activate |
|----------|------------|-----------------|
| Instagram | Paid Partnership label | Settings → Creator → Branded Content |
| TikTok | Paid promotion toggle | Post settings → Disclose content |
| YouTube | Paid promotion checkbox | Video details → Paid promotion |
| Twitter/X | No native tool | Manual #ad in tweet |
| LinkedIn | No native tool | Manual disclosure in post |
